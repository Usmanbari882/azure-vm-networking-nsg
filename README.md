# Azure VM Networking Configuration

## Project Overview

In this project, I explored and configured Azure Networking concepts using a Virtual Machine environment.

I learned how Azure Virtual Machines communicate with the internet using Virtual Network, Subnet, Network Interface, Public IP and Network Security Groups.

## Azure Services Used

- Microsoft Azure Virtual Machine
- Virtual Network (VNet)
- Subnet
- Network Interface (NIC)
- Network Security Group (NSG)
- Public IP Address
